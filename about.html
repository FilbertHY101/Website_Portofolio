Tentang Saya

Perkenalkan saya Filbert Hardy Yahya. Teman-teman memanggil saya Filbert atau Hardy. Saya lahir di Palembang tahun 2002 dan saya sekarang berumur 18 tahun. Saat ini saya adalah mahasiswa di Kampus Multi Data Palembang dengan Program Studi Sistem Informasi. Pernah bersekolah di SMA Xaverius 4 Palembang dengan jurusan IPS. Saya sangat tertarik dengan dunia Teknologi Informasi oleh karena itu saya sangat senang mempelajarinya dan sangat bermanfaat untuk pendidikan yang saya tekuni sekarang ini.

Website ini adalah tempat yang saya gunakan untuk membagikan informasi tentang diri saya. Selain itu, saya juga ingin berbagi ilmu dan pengalaman. Walaupun tidak punya banyak ilmu dan pengalaman. Setidaknya bisa berkontribusi dan berbagi hal positif untuk semua. Tidak menutup kemungkinan akan ada beberapa perubahan yang saya akan lakukan untuk website ini. Semoga website ini dapat bermanfaat untuk semua. Terima kasih ya telah berkunjung kemari.. Maju terus programmer Indonesia!
